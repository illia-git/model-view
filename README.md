
# Model - View - Controller
## Examples how to implement MVC on server and client sides



## Run Locally

Clone the project

```bash
  git clone https://github.com/illia-git/model-view.git
```
### Front
Go to the project directory

```bash
  cd mvc-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
### Node
Go to the project directory

```bash
  cd mvc-node
```

Install dependencies

```bash
  npm install
```

Start the server (ru)

```bash
  npm run start
```
